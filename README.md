## Dependencies
- cmake
- raylib
- C compiler (gcc/clang).

## Setup and Build

1. Clone the repository
```bash
git clone https://github.com/ivanhue/loro.git
cd loro
```
2. Run the script
```bash
chmod +x run.sh
./run.sh
```
3. Add configuration
```bash
mkdir -p ~/.config/loro
cp themes/base.ini ~/.config/loro/config.ini
```
