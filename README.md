## Dependencies
- cmake
- raylib
- C compiler (gcc/clang).

## Setup and Build

1. Clone the repository
```bash
git clone https://github.com/ivanhue/loro.git
cd loro
```
2. Run the script
```bash
chmod +x run.sh
./run.sh
```
